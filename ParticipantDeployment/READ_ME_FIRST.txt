========================================
FOR RESEARCH PARTICIPANTS
========================================

Thank you for participating in this study!

This program monitors your coding behavior to study programming stress.
It is completely safe and respects your privacy.

========================================
WHAT IT DOES
========================================

✓ Tracks typing speed and pauses
✓ Detects when you compile your code
✓ Calculates a stress score (0-100)
✓ Saves data every 30 seconds

✗ Does NOT record your actual code
✗ Does NOT log what you type
✗ Does NOT send data over the internet

========================================
HOW TO USE
========================================

1. START DATA COLLECTION
   → Double-click: START_CLEAN_COLLECTOR.bat
   → A black window will open
   → Keep it open while you code

2. WORK NORMALLY
   → Open CodeBlocks
   → Code as you normally would
   → The monitor runs silently in the background

3. WHEN FINISHED
   → Click the black window
   → Press Ctrl + C to stop
   → You can close everything

========================================
WHAT YOU'LL SEE
========================================

The black window shows updates like this:

[ROW  5] Keys: 234 | WPM: 32.15 | Anxiety: 45 (MODERATE) | Focus: ✓

- Keys: How many keys you've pressed
- WPM: Your typing speed (words per minute)
- Anxiety: Stress score (0-100)
- Focus: ✓ if CodeBlocks is active, ✗ if not

========================================
WHERE DATA IS SAVED
========================================

Location: Documents\AnxietyMonitorData\
File: data_YYYYMMDD.csv (one file per day)

DO NOT open the CSV file while the monitor is running.
The researcher will collect this file at the end of the session.

========================================
PRIVACY GUARANTEE
========================================

This monitor ONLY collects:
- Timing information (when you type, not what)
- Counts (how many keystrokes, not which keys)
- Window focus (are you in CodeBlocks?)

This monitor DOES NOT collect:
- Your source code
- Your file contents
- What keys you press
- Your personal information

All data stays on YOUR computer.
No internet connection is used.

========================================
FREQUENTLY ASKED QUESTIONS
========================================

Q: Will this slow down my computer?
A: No. It uses less than 1% of CPU power.

Q: What if I need a break?
A: Stop the monitor (Ctrl+C), take your break, then restart it.

Q: Can I see what data is being collected?
A: Yes! After stopping, you can open the CSV file to review.

Q: What if I close the black window by accident?
A: Double-click START_CLEAN_COLLECTOR.bat again to restart.
   It will continue in the same daily file.

Q: What happens if my computer crashes?
A: All data up to the last 30-second save is safe in the CSV file.

========================================
TROUBLESHOOTING
========================================

Problem: "Python is not recognized"
→ Python is not installed. Contact the researcher.

Problem: "PermissionError" when starting
→ Close the CSV file if you have it open in Excel.

Problem: Nothing happens when I type
→ Make sure CodeBlocks is the active window.
→ Check that Focus shows: ✓ in the monitor window.

Problem: The numbers don't change
→ This is normal if you're not typing.
→ Numbers update every 30 seconds during active coding.
